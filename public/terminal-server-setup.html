üîß Troubleshooting

Port Already in Use

If port 3001 is already in use, you can change it:

# Kill existing process
pkill -f terminal-server

# Or use different port (edit terminal-server.cjs)
const PORT = 3002;

Permission Denied

Make sure the file has execute permissions:

chmod +x terminal-server.cjs

Node.js Not Found

Install Node.js from nodejs.org and restart your terminal
